Este repositório foi criado para armazenar e organizar todos os exercícios, projetos e atividades desenvolvidos durante o curso de Python. O objetivo é registrar minha evolução no aprendizado da linguagem, reunindo exemplos práticos sobre lógica de programação, estruturas de decisão, estruturas de repetição, funções, manipulação de dados e outros conteúdos estudados ao longo do curso. Além de servir como portfólio de aprendizado, este repositório facilita a consulta e revisão dos exercícios realizados.
